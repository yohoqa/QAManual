# Тестирование веб и мобильной версии приложения Яндекс.Самокат

###  Постановка задачи:

На основе требований к приложению и макетов в Figma, провести:

1. Тестирование веб-приложения Яндекс.Самокат
    - Вкладка «Задание 1: **чек-лист**» - составить чек-лист по требованиям к экрану «Статус заказа»  
    - Вкладка «Задание 1: **данные валидации**» - составить проверки на валидацию полей для экрана «Сделать заказ»  
    - Вкладка «Задание 1: **баги вне тестовой документации**» - баги, найденные вне тестовой документации 

2. Тестирование мобильного приложения Яндекс.Самокат v2.0
    - Вкладка «Задание 2: **тест-кейсы**» - спроектировать тест-кейсы и протестировать функциональность, написать кейсы на вёрстку по макетам к этой функциональности

3. Тестирование API Яндекс.Самокат v1.0.0
    - Вкладка «Задание 3: **чек-лист**» - в чек-листе должны быть результаты выполнения тестов и заведённые баг-репорты.
